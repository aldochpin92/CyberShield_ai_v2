# CyberShield AI — El Guardián 🛡️
### Latin Passion, Cyber Strength. 

**CyberShield AI** is a specialized, multilingual security intelligence agent designed for the **2026 FIFA World Cup**. Developed as a joint collaboration between **Aldo (Creator)**, **Copilot (Architecture)**, and **Gemini (Intelligence)**, this project bridges the gap between high-tech SOC (Security Operations Center) capabilities and protective "street smarts."

## 🦅 The Vision: El Águila
The CyberShield logo represents the fusion of vision and defense. 
* **The Eagle:** Sharp, swift threat detection.
* **The Shield:** Unyielding protection for fans, travelers, and infrastructure.
* **The Circuitry:** AI-driven intelligence operating at the speed of light.

## 🚀 Core Capabilities
* **Linguistic Defense:** Real-time analysis of regional slang and dialects to detect extortion, "Virtual Kidnapping," and smishing.
* **Event-Security Logic:** Specialized threat modeling for the 16 Host Cities of the 2026 World Cup.
* **Safety Playbooks:** Actionable, human-centric response plans for cybersecurity incidents (Phishing, Rogue Wi-Fi, NFC/Bluetooth Hijacking).
* **OSINT & Web Analysis:** Rapid scanning of suspicious URLs and fraudulent social profiles targeting tournament attendees.

## 🛠️ Technical Stack
* **AI Intelligence:** Google Gemini (Custom System Prompting for El Guardián Identity).
* **Backend:** Python 3.14 + FastAPI (High-concurrency threat processing).
* **Frontend:** Cyber-themed Dashboard (Electric Blue & Gunmetal Silver UI).
* **Environment:** Secure .env integration for Global API key management.
